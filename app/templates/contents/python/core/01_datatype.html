딕셔너리 정렬

리스트와 달리 딕셔너리(객체)는 .sort() 메소드를 직접 가지고 있지 않습니다.
대신, 파이썬 내장 함수인 sorted()를 사용하여 딕셔너리의 내용을 정렬할 수 있습니다.
sorted() 함수는 키, 값 또는 키-값 쌍(아이템)을 기준으로 정렬된 새로운 리스트를 반환합니다.

b = {'name': 'pey', 'phone': '0119993323', 'birth': '1118'}

키(key)를 기준으로 정렬하기

sorted() # 리턴값은 정렬된 리스트

# b.items()를 키를 기준으로 정렬합니다.
sorted_items_by_key = sorted(b.items())
print(sorted_items_by_key)
# 출력: [('birth', '1118'), ('name', 'pey'), ('phone', '0119993323')]
값(value)을 기준으로 정렬하기

값으로 정렬하려면 sorted() 함수의 key 인자를 사용해야 합니다. lambda 함수를 사용하면 간단하게 정렬 기준을 지정할 수 있습니다. key=lambda item: item[1]은 각 아이템(키-값 쌍 튜플)에서 인덱스 1에 해당하는 값(value)을 기준으로 정렬하라는 의미입니다.

# b.items()를 값을 기준으로 정렬합니다.
sorted_items_by_value = sorted(b.items(), key=lambda item: item[1])
print(sorted_items_by_value)
# 출력: [('birth', '1118'), ('phone', '0119993323'), ('name', 'pey')]

# 값을 기준으로 내림차순 정렬하려면 reverse=True를 추가합니다.
sorted_items_by_value_desc = sorted(b.items(), key=lambda item: item[1], reverse=True)
print(sorted_items_by_value_desc)
# 출력: [('name', 'pey'), ('phone', '0119993323'), ('birth', '1118')]
