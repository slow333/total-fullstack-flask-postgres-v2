if name == “main”의 의미

//mod1.py
def add(a,b):
    return a+b
def sub(a,b):
    return a-b
add(3,4)
sub(4,2)

위와 같이하면 import 시 자동으로 실행됨
shell에서 import mod1을 하면 add(3,4)와 sub(4,2)가 실행됨
이를 방지하기 위해

//mod1.py
def add(a,b):
    return a+b
def sub(a,b):
    return a-b
if __name__ == "__main__":
  add(3,4)
  sub(4,2)

위와 같이 하면 shell에서 python mod1.py를 하면 if name == “main” 이 참이되어 실행되고,
import mod1을 하면 실행되지 않음

PI = 3.141592
class Math:
    def solv(self, r):
        return PI * (r ** 2)

    return a+b

import mod2
